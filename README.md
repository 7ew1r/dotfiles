## zsh
### required
- peco
- ghq
- coreutiles
- Powerline (optional)

Add the following on `~/.zshenv`

``` shell
export ZDOTDIR=$HOME/.zsh
```

$ cp $HOME/dotfiles/.zsh/.zshrc_local.temp $HOME/dotfiles/.zsh/.zshrc_local
$ vi $HOME/dotfiles/.zsh/.zshrc_local
add local setting here
